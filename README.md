# coc-txt-hover

doHover for txt to look up a word online.

## Install

`:CocInstall coc-txt-hover`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-txt-hover-keymap)`

## Lists

`:CocList demo_list`
